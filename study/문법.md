### 기본
* Start() : 처음 한번 실행
* Update() : FPS마다 실행
* Debug.Log() : 로그

### 바로사용 가능

* tracsform (Object 위치 정보)
```
traceform.position
```

* Vector3
  * 3D로 표현
  ```
  Vector3.left = (-1,0,0)
  ```
 
* Time.deltaTime
  * FPS(프레임)를 맞춰주는 역할을 한다

* Instantiate(Object)
 * 매 FPS마다 Object를 찍어낸다

* Destroy(Object, sec);
 * Object를 sec초 후에 지운다.
 * 메모리관리

### 선언후 사용 가능 

* Rigidbody2D
  * Object에 물리법칙을 주는 역할
```
점프 예제
2D 물체에 velocity를 주는 예제
public float jumpPower;
void Start(){
    rb = GetComponent<Rigidbody2D>();
}
void Update(){
    if (Input.GetMouseButtonDown(0)){
        rb.velocity = Vector2.up * jumpPower; // (0,x)
    }
}
```
